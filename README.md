# Math gym

Simple application for practicing multiplications

## Getting Started

```bash
npm install
npm start
```

## Running the tests

To run the unit tests for the custom elements, run:

```bash
npm test
```

## Building for production

To build the application for production, run:

```bash
npm run build
```
